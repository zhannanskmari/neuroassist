# 🧠 Проект: Нейро-ассистент эксперта по мышлению

## 1. Паспорт проекта

| Параметр | Описание |
| :--- | :--- |
| **Название** | Нейро-ассистент эксперта по мышлению |
| **Роль бота** | Интеллектуальный помощник, проводящий первичную диагностику стиля мышления и записывающий клиентов на сессию к эксперту. |
| **Ключевая цель** | Конвертация входящего запроса в подтвержденную запись на онлайн-встречу (90 мин) с синхронизацией в Google Календарь. |
| **Платформа реализации** | Telegram-бот (на базе Suvvy.ai). |
| **Ядро интеллекта** | LLM Qwen (для генерации промптов и обработки естественного языка). |

---

## 2. Функциональные возможности

### 🗣 Осмысленное ведение диалога
Бот не просто отвечает по шаблону, а ведет контекстный диалог, имитирующий живое общение с ассистентом.
*   **Сценарий диагностики:** Последовательно задает 5–7 уточняющих вопросов для выявления стиля принятия решений, реакции на информацию и текущих барьеров клиента.
*   **Адаптивность:** Меняет тон и глубину ответов в зависимости от реакций пользователя (сомнения, интерес, срочность).
*   **Работа с возражениями:** Мягко отрабатывает типичные возражения («дорого», «нет времени», «не верю в ИИ») используя заложенные скрипты.

### 📚 Ответы на вопросы по базе знаний
Бот выступает как первая линия поддержки, снимая нагрузку с эксперта.
*   **Источники данных:** База знаний сформирована на основе методологии эксперта, описания услуг, этических принципов и FAQ.
*   **Тематика ответов:**
    *   Суть методики и роль ИИ в процессе.
    *   Вопросы конфиденциальности и безопасности данных.
    *   Формат, длительность и стоимость встречи.
    *   Гарантии и пост-сопровождение.
*   **Ограничения:** Бот запрограммирован не давать медицинских диагнозов и не подменять работу психотерапевта.

### 📅 Запись на встречу и интеграция с Google Календарь
Автоматизация административной части воронки продаж.
1.  **Выбор слота:** Предложение клиенту доступных временных окон из расписания эксперта.
2.  **Сбор контактов:** Фиксация имени и номера телефона (с объяснением цели использования).
3.  **Оплата:** Отправка ссылки на оплату услуги.
4.  **Синхронизация:** После подтверждения оплаты бот автоматически создает событие в **Google Календаре** эксперта и отправляет клиенту ссылку на видеовстречу (Zoom/Telegram) и файл `.ics` для добавления в личный календарь.
5.  **Напоминания:** Автоматическая отправка напоминаний за 24 часа и за 1 час до встречи.

## 3. Технологический стек

### 🤖 Qwen (Large Language Model)
Использовался как фундаментальная языковая модель для решения задач высокой сложности:
*   **Генерация системного промпта:** Создание детальной инструкции для бота, включающей роль, стиль общения (дружелюбный эксперт), ограничения (запрет на термин «когнитивные процессы», запрет на анализ документов) и сценарии поведения.
*   **Формирование базы знаний:** Структурирование информации о продукте, методике и ответах на 50+ сложных вопросов клиентов в формат, оптимальный для поиска (RAG).
*   **Настройка тональности:** Калибровка ответов так, чтобы они звучали естественно, эмпатично и профессионально, избегая канцеляризмов и «роботизированности».

### ⚙️ Suvvy.ai (Платформа разработки)
Сервис-конструктор, на котором развернут и функционирует бот:
*   **Оркестрация диалога:** Управление состояниями диалога (приветствие → диагностика → презентация → запись).
*   **Интеграции:** Настройка вебхуков для связи с Google Calendar API и платежными шлюзами.
*   **Интерфейс:** Реализация удобных кнопок (Inline Keyboard) для выбора опций диагностики и времени встречи.
*   **Хостинг:** Обеспечение бесперебойной работы бота 24/7.

## 5. Этические принципы и ограничения

В промпт бота жестко внедрены следующие правила:
*   **Терминология:** Запрещено использование сложного термина «когнитивные процессы». Используются замены: «стиль мышления», «привычки принятия решений».
*   **Документы:** Бот не запрашивает и не анализирует файлы/документы клиентов. Работа ведется только в формате диалога.
*   **Медицина:** При признаках депрессии или выгорания бот рекомендует обратиться к профильному специалисту (психологу/врачу), не беря на себя функции лечения.
*   **Конфиденциальность:** Гарантия удаления данных по запросу клиента и отсутствие передачи информации третьим лицам.

## 6. Ожидаемые результаты внедрения

1.  **Рост конверсии:** За счет мгновенной реакции и персонализированной диагностики на этапе прогрева.
2.  **Экономия времени эксперта:** До 80% рутинных вопросов и процесс записи полностью автоматизированы.
3.  **Качество лидов:** Клиенты приходят на встречу уже «прогретыми», прошедшими первичный анализ и понимающими ценность услуги.
4.  **Отсутствие человеческих ошибок:** Исключены двойные записи и потерянные сообщения благодаря интеграции с календарем.
